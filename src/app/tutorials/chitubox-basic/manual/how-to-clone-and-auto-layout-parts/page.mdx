# How to Clone and Auto-layout parts?

Step 1. Launch CHITUBOX software and import your part.

<ImageContainer>
	<AntdImage
		alt="launch CHITUBOX Basic"
		src={
			"/en/academy/images/docs/tutorials/chitubox-basic/manual/how-to-clone-and-auto-layout-parts/launch-chitubox-basic.jpg"
		}
	/>
</ImageContainer>

Step 2. Select your part and then click “Clone Current Model” button to clone it. The part will be cloned once you click the button.

<ImageContainer>
	<AntdImage
		alt="Copy feature in CHITUBOX"
		src={
			"/en/academy/images/docs/tutorials/chitubox-basic/manual/how-to-clone-and-auto-layout-parts/copy-feature-in-chitubox.jpg"
		}
	/>
</ImageContainer>

<ImageContainer>
	<AntdImage
		alt="copy model"
		src={
			"/en/academy/images/docs/tutorials/chitubox-basic/manual/how-to-clone-and-auto-layout-parts/copy-model.gif"
		}
	/>
</ImageContainer>

Step 3. Click “Auto Layout” button to layout parts by center, X side or Y side. You can also customize the min distance betweem parts.
